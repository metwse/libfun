Detailed Description

Basic hashmap.

hashmap uses the FNV hash function and open addressing method. It stores data as key-value pairs, allowing variable-length keys while storing fixed-length values.

Removes the key-value pair from the hashmap, returns a pointer to the value.

Attention
The returned value pointer is valid until the next hashmap operation. If the user wants to continue using the value, they must copy underlying data.

The key parameter must be null-terminated.
